# スカベンジャー v2 ブラッシュアップ計画

## 現状 (934行)
- BSPダンジョン生成、ターン制、FOV、バンプ戦闘、テトリス鞄、拠点/ショップ
- 基本的なゲームループは動く

## ブラッシュアップ (ユーザーフィードバック反映済み)

### Phase 1: 鞄・アイテムUX (最優先 — コア体験に直結)

- [ ] **Task 1: アイテム形状の事前プレビュー**
  - 床のアイテムを踏んだ時、拾う前にアイテムの形状・名前・効果・売値を表示するモーダル
  - 「拾う」「無視」ボタン → 「無視」ならアイテムは床に残ったまま
  - ルートバッグのアイテムも同様に、選択時に形状プレビュー表示
  - 敵ドロップのルートバッグでも各アイテムの形状が見えるように

- [ ] **Task 2: 鞄配置後も開いたまま & ステータスバー常時表示**
  - アイテムを鞄に配置した後、鞄画面を閉じないで開いたまま維持
  - 続けて配置位置の確認や整理ができるように
  - 鞄オーバーレイの背景を半透明にして、裏のマップがうっすら見える
  - **ステータスバー(HP, 満腹度, 階数等)はオーバーレイの上にも常時表示**
  - ターン進行で敵に攻撃された場合、ダメージ表示が見えるように

- [ ] **Task 3: アイテム再配置（移動）機能**
  - 鞄内の設置済みアイテムをタップ → 「移動」「使う」「捨てる」メニュー表示
  - 「移動」選択 → アイテムが一時的に外れて再配置モード（緑/赤プレビュー付き）
  - 移動完了でもターン1消費
  - キャンセルしたら元の位置に戻る（ターン消費なし）

- [ ] **Task 4: アイテム効果説明**
  - 鞄内のアイテムタップ時に詳細パネル表示:
    - 名前、アイコン、形状プレビュー（小さいグリッド）
    - 効果テキスト: 「HP+20回復」「満腹度+30」「攻撃力+7」「売値: 40G」
    - レアリティ表示
  - アイテム一覧で各アイテムのツールチップ的な情報
  - 武器の場合: 「装備中 → 攻撃力+7」のように現在のボーナスも表示

- [ ] **Task 5: 配置プレビュー（緑/赤）**
  - グリッドセルにタッチ/ホバーでアイテム形状の緑(配置可)/赤(配置不可)プレビュー
  - 配置対象アイテムの形状をセル上にオーバーレイ表示
  - アイテムのレアリティに応じたボーダー色（common=#aaa, uncommon=#4caf50, rare=#2196f3, legendary=#ff9800）

### Phase 2: ゲームシステム修正

- [ ] **Task 6: レベルリセット（シレン式）**
  - 探索に入るたびにレベル1に戻る
  - レベルアップのHP/攻撃/防御ボーナスはダンジョン内でのみ有効
  - 拠点に戻ると基礎ステータスに戻る
  - 装備（鞄の中の武器）のボーナスは維持
  - これによりダンジョン序盤の緊張感が毎回ある

- [ ] **Task 7: アイテムのドロップ/残留修正**
  - 床のアイテムを踏んで「無視」した場合、アイテムはその場に残る（現在は拾った時点で消える）
  - 鞄からアイテムを「捨てる」と、プレイヤーの足元に落ちる（現在も実装済みだが確認）
  - 鞄が一杯で配置できなかった場合、アイテムを床に戻すオプション

- [ ] **Task 8: D-pad二重入力防止**
  - タッチ後150msのデバウンス（クールダウン）
  - click と touchstart の両方で発火してる問題を修正
  - 連打時は最初の入力のみ受け付け、150ms後に次を受け付ける

### Phase 3: 拠点の充実

- [ ] **Task 9: 自宅と収納箱**
  - 拠点に「🏠 自宅」セクション追加
  - 収納箱: アイテムをリスト形式で保管（テトリス配置なし）
  - 容量上限: 20スロット（1アイテム = そのアイテムのマス数分のスロットを消費）
    - 例: 包帯(1×1)=1スロット、ナイフ(1×3)=3スロット、金塊(2×3)=6スロット
  - 収納箱 ↔ 鞄 のアイテム移動
  - 回復アイテムや武器を保管して次の探索に持っていける
  - 収納箱の容量拡張はショップで購入可能

- [ ] **Task 10: 拠点の機能拡張**
  - 拠点画面のレイアウト改善（タブ or セクション分け）
  - 「🏠 自宅」— 収納箱、休憩（HP全回復）
  - 「🛒 ショップ」— アイテム売却、アップグレード購入
  - 「📊 記録」— 探索統計、最深階、倒した敵数
  - 拠点にいる間のBGM的な雰囲気（安全感）
  - 「⚔️ 探索開始」ボタンを目立つ位置に

### Phase 4: マップ・探索

- [ ] **Task 11: ミニマップ**
  - 画面右上に小さいミニマップ（48×36px程度）
  - 踏破（seen）した部分のみ表示
  - 壁=暗い色、床=明るい色、プレイヤー=白い点
  - 階段、出口の位置も小さく表示
  - タップで拡大/縮小

- [ ] **Task 12: ログ強化・敵HPバー**
  - ログを1行→3行表示（直近3件スクロール）
  - 敵の下にHP割合バー（小さい赤バー）
  - ダメージ数値を大きく表示
  - レベルアップ時の画面フラッシュ演出

### Phase 5: ゲーム性の深み

- [ ] **Task 13: 敵AI特殊行動**
  - ネズミ: 2倍速（1ターンに2回行動）
  - チンピラ: アイテム窃盗（隣接時に鞄から1個ランダムに奪う）→倒すとドロップ
  - 盗賊: 遠距離攻撃（2マス先から射撃、ダメージは通常の半分）
  - 警備ロボ: 壁を壊して通路を作る

- [ ] **Task 14: マップ多様性**
  - 宝物部屋（アイテム多め、敵も多め）
  - 罠タイル: 踏むとダメージ or 満腹度減少
  - コンテナ（箱、ロッカー）: マップ上の開けられるオブジェクト → ルート画面
  - 部屋のフレーバーテキスト

- [ ] **Task 15: ダッシュ移動**
  - D-pad長押し(500ms以上)で自動移動開始
  - 壁、敵、アイテム、分岐点で自動停止
  - 各マス移動ごとに1ターン消費（高速描画）

### Phase 6: リプレイ性

- [ ] **Task 16: 統計・実績**
  - 累計統計: 最深階到達、倒した敵数、拾ったアイテム数、総売上、探索回数
  - 実績: 「最初の敵を倒した」「5階到達」「金塊を持ち帰った」「全エリアクリア」

- [ ] **Task 17: チュートリアル**
  - 初回プレイ時のガイド表示（移動、攻撃、アイテム拾い、鞄操作）
  - 「？」ボタンでいつでもヘルプ
