# ダンジョン商人 改善リスト

各項目を1つずつ実装していく。完了したら ✅ マークをつける。

## 改善項目

1. [x] **チュートリアル追加** — 初回プレイ時にステップバイステップで操作説明（ギルド→クエスト→施設の流れ）
2. [x] **BGM/SE追加** — Web Audio APIで環境音・ボタン押下音・収穫音などの効果音
3. [x] **季節システム** — 春夏秋冬（各7日）で育てられる作物が変わる。冬は畑が使えない
4. [x] **冒険者の個性** — 名前だけでなくランク・得意分野を持つ。得意分野のクエストは日数短縮
5. [x] **クエスト結果の詳細化** — 成功/失敗判定、失敗時は報酬の一部返却
6. [x] **町のビジュアル** — 町タブにシンプルな町の俯瞰図（施設が建つと絵が変わる）
7. [x] **実績システム** — 「初めてのクエスト」「100G稼いだ」などの実績バッジ
8. [x] **鍛冶・醸造所** — 素材を加工して高価な商品を作る施設（剣、ポーション等）
9. [x] **天気システム** — 晴れ/雨/嵐がランダムに変化。雨の日は水やり不要、嵐は作物ダメージ
10. [x] **市場価格変動** — クエスト報酬の相場が日ごとに微変動。需給バランス
11. [ ] **倉庫容量制限＆拡張** — 素材に上限を設けて倉庫拡張を建設要素に
12. [ ] **冒険者ギルドランク** — ギルドの評判でランクが上がり、高難度クエストが解禁
13. [ ] **ランチの品質表示** — 牧場の動物の状態をもっと視覚的に（満腹度アイコン等）
14. [ ] **夜/昼サイクル演出** — 背景色が時間帯で変わる演出
15. [ ] **統計・グラフ** — 収支推移をグラフ表示（直近10日分）
