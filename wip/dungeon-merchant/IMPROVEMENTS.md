# ダンジョン商人 改善リスト

各項目を1つずつ実装していく。完了したら [x] マークをつける。

## 完了済み

1. [x] **チュートリアル追加** — 初回プレイ時にステップバイステップで操作説明
2. [x] **BGM/SE追加** — Web Audio APIで環境音・ボタン押下音・収穫音
3. [x] **季節システム** — 春夏秋冬（各7日）で育てられる作物が変わる
4. [x] **冒険者の個性** — ランク・得意分野を持つ
5. [x] **クエスト結果の詳細化** — 成功/失敗判定
6. [x] **町のビジュアル** — 町タブにシンプルな町の俯瞰図
7. [x] **実績システム** — 実績バッジ
8. [x] **鍛冶・醸造所** — 素材を加工して高価な商品を作る施設
9. [x] **天気システム** — 晴れ/雨/嵐がランダムに変化

## 優先改善（次回から順に実装）

10. [x] **経済バランス修正** — 宿泊収入を最低限に調整（1人あたり3~5G程度）。食事・製品販売の収入を大幅UP。素材→加工品→販売が主な稼ぎ手段になるバランスに。宿泊だけでは赤字で回らない設計にする。調理収入は素材費の2~3倍。鍛冶品は5倍以上の利益。

11. [x] **チュートリアル強化** — 既存のチュートリアルを改善：説明だけでなく、実際に「木材採取クエスト掲載→宿屋建設」まで一緒にやるインタラクティブ形式に。チュートリアルスキップ時も、画面上部に「まずはギルドで木材を集めて宿屋を建てよう！」という案内バナーを表示。宿屋建設完了まで案内を出し続ける。

12. [x] **食堂・鍛冶を個別タブ化** — 現在「施設」タブの中にある食堂と鍛冶を、農場のように独立したメインタブにする。タブ構成：⚔️ギルド | 🏡農場 | 🍳食堂 | 🔨鍛冶 | 📦倉庫 | 🏘️町。食堂タブで調理、鍛冶タブで武器・ポーション制作ができるように。施設タブは宿屋・厩舎など建設系のみ残す。

13. [ ] **作り置き＆自動販売システム** — 食堂で作った料理や鍛冶品を「在庫」として保持。冒険者がクエスト出発時に自動で食事を購入（在庫から消費→Gold加算）。宿泊中の冒険者も毎日自動で食事を買う。在庫がなければ買わない。料理の在庫数と売値を表示。鍛冶品も同様に冒険者が自動購入（武器→クエスト日数短縮、ポーション→成功率UP）。

14. [x] **市場価格変動** — クエスト報酬の相場が日ごとに微変動。需給バランス
15. [x] **倉庫容量制限＆拡張** — 素材に上限を設けて倉庫拡張を建設要素に
16. [ ] **冒険者ギルドランク** — ギルドの評判でランクが上がり、高難度クエストが解禁
17. [ ] **牧場の動物ビジュアル改善** — 満腹度アイコン等
18. [ ] **夜/昼サイクル演出** — 背景色が時間帯で変わる演出
19. [ ] **統計・グラフ** — 収支推移をグラフ表示（直近10日分）
