# ラーメン屋台物語 - カイロ風リメイク設計書

## コンセプト
カイロソフトの「投資→フィードバック→次の投資」ループをラーメン屋経営に落とし込む。
店主視点で「自分のラーメンは客に響いてるか？」を常に感じられるゲーム。

## 最重要原則：プレイヤーの自由

**ラーメン屋のゴールはプレイヤーが決める。**
ゲーム側が「屋台→小店舗→大店舗」と一本道を強制しない。

### プレイヤーが選べる経営スタイルの例
- **薄利多売型**: 安いラーメン（500円）を大量に捌く。席を増やし、寸胴を増やし、回転率勝負
- **高級路線型**: こだわりの一杯（1500円）。席は少ない、少数精鋭、リピーター重視
- **バランス型**: 中価格帯でメニュー豊富。幅広い客層
- **研究特化型**: ひたすらスープ研究。「究極の一杯」を追求

### ゲームのゴール
- **終わりはない**（長く続けられるほど良い）
- スコア指標:
  - 累計利益（総売上 - 総経費）
  - 連続営業日数（赤字が続くと閉店のリスク）
  - 食べログスコア（口コミ評価）
- 閉店条件: 所持金が0以下の状態で3日連続赤字 → ゲームオーバー
  - ただし借金してでも続けるオプションもあり（利子付き）

## 店のレイアウト

**プレイヤーが自由に配置する。**
店のサイズ自体も拡張可能（お金で土地を買い増す）。

### 配置可能なもの
| カテゴリ | アイテム | コスト | 効果 |
|---------|---------|------|------|
| 席 | カウンター席 | 200円 | 1席、回転早い |
| 席 | テーブル席 | 500円 | 2席、滞在長いがチップ多い |
| 席 | 座敷席 | 800円 | 4席、家族連れ向き |
| 厨房 | 寸胴（小） | 500円 | スープ生産 基本 |
| 厨房 | 寸胴（大） | 2000円 | スープ生産 2倍 |
| 厨房 | 製麺機 | 3000円 | 自家製麺で品質UP |
| 厨房 | 冷蔵庫 | 1000円 | 食材保存量UP |
| 外装 | 看板（小） | 300円 | 集客+5% |
| 外装 | 看板（大） | 1500円 | 集客+15% |
| 外装 | のぼり | 200円 | 集客+3% |
| 内装 | 観葉植物 | 200円 | 雰囲気UP |
| 内装 | テレビ | 500円 | 待ち時間の不満軽減 |

### 隣接コンボ（カイロ式）
- 寸胴 + 製麺機 = 「こだわり工房」(品質+15%)
- カウンター席 + 寸胴 = 「ライブ感」(満足度+10%)
- テーブル席 + 観葉植物 = 「癒し空間」(チップ+20%)
- 座敷席 + テレビ = 「くつろぎ処」(滞在延長→追加注文)
- 看板 + のぼり = 「目立つ店」(集客+25%)
- 冷蔵庫 + 製麺機 = 「鮮度抜群」(品質+10%)

### 店のサイズ拡張
- 初期: 6×5グリッド（最下行は道路）
- 拡張1: 8×5 (+5000円)
- 拡張2: 10×6 (+20000円)
- 拡張3: 12×7 (+80000円)
- プレイヤーは拡張するかしないかを選べる（小さい店のままでもOK）

## スープ研究

### 研究システム
- ダシ素材を組み合わせてスープ開発
- パラメータ: コク、キレ、香り、濃さ（0~100）
- 各素材がパラメータに影響
- 同じ素材でも煮込み時間で変化

### ダシ素材（研究で段階的に解放）
| 素材 | コク | キレ | 香り | 濃さ | 解放条件 |
|------|-----|-----|-----|-----|---------|
| 鶏ガラ | +20 | +10 | +5 | +15 | 初期 |
| 豚骨 | +30 | +5 | +5 | +30 | 研究10pt |
| 煮干し | +10 | +25 | +20 | +10 | 研究15pt |
| 昆布 | +5 | +15 | +25 | +5 | 研究10pt |
| 野菜 | +10 | +10 | +15 | +5 | 初期 |
| 鰹節 | +15 | +20 | +30 | +10 | 研究25pt |
| 貝類 | +20 | +15 | +20 | +15 | 研究40pt |

### 麺
| 種類 | 特徴 | 相性の良いスープ |
|------|------|----------------|
| 極細ストレート | 回転早い、繊細 | キレ高いスープ |
| 中細ストレート | バランス | なんでも合う |
| 太麺ちぢれ | スープ絡む | コク高いスープ |
| 極太麺 | 食べ応え | 濃さ高いスープ |
| 全粒粉麺 | 風味豊か | 香り高いスープ |

### タレ
- 醤油（初期）
- 味噌（研究10pt）
- 塩（研究15pt）

### トッピング
- 段階的に解放（研究ptで）
- 各トッピングが味のパラメータに微調整を加える
- 原価に影響

## 客の個性と感情

### 客タイプ（評判で出現率変化）
| タイプ | 予算 | 好み | 特徴 |
|-------|-----|-----|------|
| サラリーマン | 600~1000円 | こだわりなし | 回転早い、昼に多い |
| 学生 | 400~700円 | 量重視 | 安さ重視、大盛り好き |
| グルメ客 | 800~2000円 | 品質重視 | 口コミ影響大 |
| 家族連れ | 500~1200円 | マイルド好み | 座敷好き、複数注文 |
| 常連 | 安定 | 特定メニュー | リピート、口コミ |

### 感情フィードバック
- 食後の表情: 😋(大満足) 😊(満足) 😐(普通) 😞(不満)
- タップで感想: 「スープのコクが好み！」「麺がちょっと柔らかい」「値段の割に...」
- 満足した客はリピーター候補に
- 不満な客は二度と来ない（口コミも悪い）

### リピーター
- 満足度80%以上→リピーター判定
- リピーターには名前がつく
- 好みを覚えている（「田中さん: 濃い味好き」）
- 友達を連れてくることがある（口コミの可視化）

## 口コミ・エゴサ

### 口コミタブ
- 来店した客が確率でレビューを残す
- 「★★★★☆ スープは絶品。でも待ち時間が...」
- 「★★★★★ 人生で一番うまい味噌ラーメン」
- 「★★☆☆☆ 値段に見合わない」
- グルメ客のレビューは影響力が大きい

### 食べログスコア
- 1.0~5.0 で変動
- 口コミの平均から算出
- スコアが高いほど新規客が増える

## スタッフ

### 採用
- バイト（安い、スキル低い、シフト制限あり）
- 社員（高い、成長する、安定）
- 研究ptで「求人広告」を出すと良い人材が来やすい

### スキル
- 調理速度: ラーメンを作る速さ
- 接客力: 客の満足度ボーナス
- 仕込み: スープ・麺の品質ボーナス
- 経験で成長（遅い）。教育投資で加速。

### 経費
- スタッフ人件費は毎日発生（固定費）
- 多く雇えば提供速度UP、でも赤字リスクも

## 数値フィードバック

### 日次レポート
- 売上、客数、満足度平均
- 原価率（食材費/売上）
- 人件費、家賃
- 純利益

### 投資効果の可視化
- 設備導入前後の比較: 「製麺機導入→品質+15→満足度+8%→客単価+12%」
- 直感的に「この投資は正解だった」と分かる

## イベント（ランダム）
- テレビ取材 → 翌日客殺到
- 近隣にチェーン店 → 安さ勝負の客が流れる
- 食材高騰 → 原価率悪化
- 常連の引っ越し → 寂しい
- ラーメン評論家来店 → 高評価なら口コミ爆増
- 季節イベント（夏は冷やし中華解放）

## 自動歩行する客（ビジュアル）
- 通行人が道を歩く
- 看板/評判に応じて入店確率変化
- 入店→着席→注文→食べる→表情→退店
- 全自動で回る「生態系」を眺める楽しさ
- 速度調整: 1x / 2x / 3x
